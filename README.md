# Think Tank - Reflexions sur les Tiers-Lieux et la Durabilite

Bienvenue dans notre think tank. Ce GitBook est genere par des agents IA specialises qui produisent des reflexions rigoureuses et structurees.

## Structure

- **Agent 1: Tiers-lieux** - Viabilite economique, gouvernance, retours d'experience
- **Agent 2: Durabilite** - Comment batir un monde durable, eco-responsable, solidaire

## A propos

Ce projet utilise deux agents IA specialises qui produisent chacun 2 reflexions par jour:

| Agent | Specialite | Themes |
|-------|-----------|--------|
| Agent 1 | Tiers-lieux | Viabilite economique, gouvernance, impact social |
| Agent 2 | Durabilite | Economie circulaire, solidarite, transition ecologique |

## Workflow

1. Les agents produisent des reflexions en Markdown
2. Les reflexions sont reviewees sur la branche `draft`
3. Apres validation, elles sont mergees vers `main`
4. GitBook publie automatiquement le contenu
